# Sippurim

Sippurim is a website dedicated to providing a platform for learning Hebrew, featuring songs, parallel texts, and engaging content. The site is powered by a Telegram bot that helps users learn and engage with Jewish history and culture through interactive features.

## Features
- Learn Hebrew with parallel texts and songs.
- Explore Jewish history and culture.
- Interactive bot for a personalized learning experience.

## Setup
This site is hosted on GitHub Pages. No local setup is required for users. Simply visit [Sippurim.com](https://sippurim.com) to access the content.
